Operand
Programs

A Program element that wraps a Program. More...

Interpretation
An Operand is an interpretation of a Program as either containing a single operand element (a Program enclosed by braces), or being empty. All elements other than the first operand element are disregarded.
Normalization
A Program is normalized as an Operand by discarding all elements other than the first operand element.
