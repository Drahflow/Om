A Simple Programming Language
Om

The Om language is an experimental answer to the question:

What is the simplest high-level programming language possible?

Om is:

  • A concatenative, homoiconic programming and algorithm notation language with:
    • Minimal syntax containing only three elements.
    • Prefix notation in which functions manipulate the remainder of the program itself.
    • Panmorphic typing that allows programming without data types.
  • A data transfer format that is trivial to parse.
  • A C++ scripting library that can be:
    • Embedded into any C++ or Objective-C++ program.
    • Extended with new data types or operations through class derivation.
  • Unicode-correct: any UTF-8 text (without byte-order marker) defines a valid Om program.

Om is not:

  • Complete. Although the intent is to develop Om into a full-featured language, the software is currently at a very early "proof of concept" stage, requiring the addition of many operations (such as basic number and file operations) and optimizations before it can be considered useful for any real-world purpose. It has been made available in order to demonstrate the underlying concepts and welcome others to get involved in early development.
  • Stationary. Om will likely undergo significant changes on its way to version 1.0.
