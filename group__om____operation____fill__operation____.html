Om

fill
Operation

"Fills" an Expression such that none of its Operators is not followed by an Operand. More...

Input
{expression} ...
Output
{expression}
Examples
  • fill {A B{2} C} {1} {3}

    {A{1}
    B{2}
    C{3}}

Implementation
Om::Operation::FillOperation

For each Operator in an Expression that has no Operand following it, pulls the next Operand following the Expression and inserts it after the Operator.
