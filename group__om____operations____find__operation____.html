find
Operations

Finds the Operator, and its Operand mapping (if any), in a Lexicon. More...

Input
{operator} {lexicon}
Output
  • If the Operator is found:
    • If an Operand is mapped to the operator:

      {lexicon}{operator{operand}}

    • Else:

      {lexicon}{operator{}}

  • Else:

    {lexicon}{}

Om::Operations::FindOperation
