Detailed Description

Each operation implementation requires the following:

  • A static char const * GetName() function that returns a null-terminated, NFD-normalized UTF-8 string to be used as a corresponding Operator name.
  • A static void Give(Evaluation) function, which gives the implementation of the operation to the Evaluation.

Including the header for an operation will cause the operation to be added to the System automatically.

Operations that consume at least one Operand must live in memory while waiting for the Operand. These operations are derived from Operations::IncompleteOperation.
